    Accessible Web Navigator

    This project, developed during the CodeCon 2024 conference in Žilina, is a simple console application that integrates OpenAI, Playwright, and NAudio in .NET to create a tool for improving web accessibility for visually impaired users. It allows users to ask AI to summarize or ask questions about websites, with text-to-speech or speech-to-text capabilities, including basic interactions like clicking on pages and writing text.
