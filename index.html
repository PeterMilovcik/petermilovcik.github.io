Conferences

CodeCon '23 - Artificial Intelligence - Panel Discussion

I had the opportunity to participate in CodeCon 2023 in Zilina, where I took part in a panel discussion dedicated to Artificial Intelligence. During the discussion, I shared my views on the current and future state of AI, as well as some examples of using AI in radiology. Additionally, I presented practical examples of using Language Models (LLMs) during software development and software testing.

OpenSlava '23 - ChatGPT in Software Testing - Analyzing Outcomes and Implications

In this presentation, we dived into using artificial intelligence in software testing, particularly the ChatGPT model by OpenAI. By exploring experiment use cases, we unpacked how ChatGPT could be used in software testing, improving efficiency. We presented the outcomes of these cases, detailing the positive impacts and any challenges encountered along the way. Moreover, we examined the broader implications of using AI in software testing, shedding light on its potential for the future. Whether you were a seasoned software engineer or simply interested in the latest AI advancements, this talk promised valuable insights and discussion.

Generative AI - Requirements to Gherkin Language

This project utilizes Generative AI to transform requirement text into Gherkin language scenarios. By leveraging advanced natural language processing techniques, it can interpret the requirements and generate corresponding Gherkin scenarios. This not only streamlines the process of writing acceptance tests in Behavior-Driven Development (BDD), but also ensures that the tests accurately reflect the intended functionality of the software.
